選んだ保存先へ自動で送る

SQLite / Parquet / MongoDB / Elasticsearch / BigQuery / S3 など、お客様の基盤へ。「どの帳票を・いつ・どこへ」を決めるだけ。

🔎

入ったかをアプリで確認
