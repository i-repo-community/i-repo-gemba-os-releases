# スクリーンショット配置先

ランディング（`/`）とユーザーマニュアルが、この `assets/img/` を共用して参照します。
以下のファイル名で PNG を置いてください（未配置の間は枠（プレースホルダ）が表示されます）:

- `screen-overview.png`    … 概要（ダッシュボード）
- `screen-connections.png` … 接続先テンプレート
- `screen-flows.png`       … フロー作成
- `screen-runs.png`        … 実行状況
- `screen-reports.png`     … データ確認
- `screen-cleanup.png`     … クリーンアップ設定
- `screen-gemba.png`       … 現場蓄積データ
- `screen-webhook.png`     … 着信 Webhook
- `screen-plugins.png`     … プラグイン
- `screen-mcp.png`         … エージェント接続（MCP）
- `screen-agent.png`       … 現場 AI チャット
- `screen-cloud.png`       … クラウド配信先の実例（GCS/S3/Azure。アプリ画面ではなく着地点）

推奨: 16:10 前後・横 1600px 程度。
