# トラブルシューティング

| 症状 | 確認すること |
|---|---|
| Windows でインストーラに警告（PCが保護されました） | SmartScreen の **「詳細情報」→「実行」** で起動できます。署名は申請手続き中です（[初回セットアップ](manual-setup.html#install-warning)参照） |
| macOS でアプリが開けない（開発元を確認できません） | アプリを**右クリック→「開く」**、または **システム設定→プライバシーとセキュリティ→「このまま開く」**（[初回セットアップ](manual-setup.html#install-warning)参照） |
| プラグインが見つからない | [プラグイン](screen-plugins.html)画面で「公式プラグインを導入」ボタンを押したか（同梱プラグインの配置） |
| 現場蓄積データに何も出ない | フローを1回成功させたか（実行状況で成功表示を確認） |
| 詳細に項目（クラスター）が出ない | フローで「詳細も一緒に送る」を有効にして送り直したか |
| 着信 Webhook が 404 | フローで「着信 Webhook を許可」したか・URL の鍵が最新か（再発行で古い URL は失効） |
| 着信 Webhook が 503 | 着信 Webhook 画面で全体が有効か・ポート番号を確認 |
| 毎日の自動実行が動かない | アプリが起動（またはトレイ常駐）しているか |
| AI エージェントがデータを読めない | [エージェント接続](screen-mcp.html)画面のコマンド／設定を登録したか・Claude Code / Codex CLI を入れたか・一度フローを成功させたか |
