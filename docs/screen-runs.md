# 実行状況

配信の実行ログ・結果・再実行を行う画面です。

> **「成功」の見分け方**：このアプリでは、**送り先が「確かに入りました」と返した確認（レシート）が取れたときだけ成功**とみなします。処理が途中で終わってもエラーにならない場合があるため、実行状況画面の成功表示で判断してください。
