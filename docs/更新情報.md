# 更新情報（What's New）

アプリの主な改善・修正をまとめます。最新の配布物は
[ダウンロード（最新リリース）](https://github.com/i-repo-community/i-repo-gemba-os-releases/releases/latest)
から入手できます。

## 2026年6月

- **配信先を種別ごとに整理して選べるように**
  送り先（プラグイン）を「データベース / データレイク / クラウドDWH / 検索 / アーカイブ」などの
  種別でグループ表示。目的の送り先を見つけやすくなりました。
- **BigQuery への配信を安定化**
  すでに用意済みのデータセットに送るとき、配信が失敗してしまうことがある問題を修正しました。
- **「プラグイン」タブの使い勝手を改善**
  表示を高速化し、各プラグインの「フォルダを開く」「アンインストール」操作を追加しました。
- **使い方ガイドを刷新**
  専門用語を整理し、よりわかりやすい説明に見直しました。仕組みの詳細は各ページ末尾の
  「技術メモ」にまとめています。
