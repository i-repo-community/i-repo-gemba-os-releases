# 接続先テンプレート

データの送り先（例: S3 バケット、MongoDB の接続先、SQLite ファイルの場所）に名前を付けて保存しておく画面です。一度登録すれば、[フロー作成](screen-flows.html)のときに選ぶだけで再利用できます。
