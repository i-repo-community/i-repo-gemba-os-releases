# 概要

アプリ全体の状況をまとめて表示するダッシュボードです。直近の配信件数・成功/失敗、つながっている送り先などをひと目で確認できます。

## はじめかたチェックリスト — 初回はここから

初めて使うときは、概要画面の先頭に **「はじめかた」チェックリスト**が出ます。「接続設定 → プラグイン導入 →
接続先を作成 → フローを作成 → 1回流して確認」の5手順の進み具合が分かり、各手順の **「案内する」**を押すと、
実際の画面をハイライトしながら順番に誘導します（ガイドツアー）。

- すべて完了する（または「非表示」）と自動で消えます。あとから見たいときは、サイドバーの **「?」** ボタンから
  はじめかたガイドや画面ツアーをいつでも開けます。
- **「AIに聞く」**からは、やりたいことを自然文で入力すると、必要な設定の場所と順番を画面上で案内してもらえます
  （どの画面でも右上の「操作をAIに聞く」からも同じことができます・読み取り専用）。
